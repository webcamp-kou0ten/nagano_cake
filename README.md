## 長野ケーキ--ECサイト--
管理者画面、顧客管理画面の両面を用意しており、ECサイトを運営するのに適した構造となっています。

## Overview
長野県にある小さな洋菓子店「ながのCAKE」の商品を通販するためのECサイト。
Instagramからの顧客流入により管理が煩雑化。それを解消することを目的とする。
